# БотыБот #
@BotyBot
## Описание ##
<font size = 4><Бот умеет проверять сайт Wildberries в поисках мужской обуви за вас и выдавать самые  актуальные предложения отсортированные по таким критериям как:*Популярность* , *Самая большая скидка* , *Рейтинг* , *Цена* , *Самое свежее* !Если вы хотите получить Топ-n товаров в выбранной вами категории, то Вам поможет команда **/find_by_criteria** . Если же хотите получать все товары по указанному Вами критерию(бренд или наименование), то Вам подойдет команда **find_top .** Для прочтения инструкции непосредственно в Боте есть команда **help**
